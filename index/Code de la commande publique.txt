Le Code de la commande publique (CCP) est un code juridique français regroupant les dispositions régissant les contrats de la commande publique, suivant une distinction entre les marchés publics et les concessions. Il est issu de l'ordonnance n° 2018-1074 du 26 novembre 2018 et du décret n° 2018-1075 du 3 décembre 2018. Entré en vigueur le 1er avril 2019, il prend la suite du code des marchés publics désormais abrogé.
